# Guide for App Admins

## Role Based Security

The app uses role based security. There are two roles and a user has to be in one of them: **volunteer** and **admin**.

You can manage the roles for the app in the Azure portal [like this](https://learn.microsoft.com/en-us/azure/static-web-apps/authentication-custom?tabs=aad%2Cinvitations#manage-roles).
